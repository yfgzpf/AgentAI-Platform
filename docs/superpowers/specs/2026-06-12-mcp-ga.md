# Spec: MCP 正式 GA (General Availability) 实装

**日期**: 2026-06-12
**作者**: AgentAI Builder
**关联 plan**: `docs/superpowers/plans/2026-06-12-mcp-ga.md`
**前置依赖**:
- `packages/agentai-gateway/src/mcp/config.ts` 已存在(占位)
- 30+ IDE/agent 平台已支持 MCP (Trae/Cursor/Copilot/Cline/Windsurf/Augment...)
- 2025-06 VS Code 1.102 MCP GA, 2025-04 Trae 加 OAuth 2.1 支持

---

## 1. 目标

把 `mcp/config.ts` 占位升级为**生产可用的 MCP 客户端**, 让:
- 用户能在设置页启用/禁用 MCP servers
- 5 个内置 servers 可直接使用: filesystem / github / slack / supabase / browser
- LLM 能调用 MCP 工具(通过 `tool-registry` 接入)
- 外部 OAuth 2.1 流程能跑(2025-04 Trae 标配)

## 2. 非目标

- ❌ 不实现 MCP server(我们是 client 端, 不发布 server)
- ❌ 不实现 stdio transport(只走 HTTP+SSE)
- ❌ 不实现 Anthropic 的 prompt caching 优化(本季跳过)
- ❌ 不动 `tools.ts` / `tool-registry.ts` 既有结构(只加 MCP bridge)

## 4. 数据模型

```typescript
// packages/agentai-gateway/src/mcp/types.ts
export interface McpServerConfig {
    id: string;                    // 唯一 ID, 例 "fs-1"
    name: string;                  // 显示名 "My Filesystem"
    type: 'filesystem' | 'github' | 'slack' | 'supabase' | 'browser' | 'custom';
    enabled: boolean;
    url: string;                   // HTTP 端点
    auth?: {
        type: 'none' | 'bearer' | 'oauth2';
        token?: string;            // bearer 用
        oauth?: OAuth2State;       // oauth2 用
    };
    capabilities?: string[];       // 拉取的 server 能力
    lastHealthCheck?: number;      // ms timestamp
    healthy?: boolean;
}

export interface OAuth2State {
    clientId: string;
    redirectUri: string;
    scope: string;
    accessToken?: string;
    refreshToken?: string;
    expiresAt?: number;
}

export interface McpToolDefinition {
    serverId: string;
    name: string;
    description: string;
    inputSchema: any;              // JSON Schema
}
```

## 6. 安全护栏

- 服务器 URL 必须 HTTPS(否则拒绝)
- OAuth client secret 不落盘(只存 token + refresh)
- Token 加密存储(用 `keys-cli` 现有 AES-256-GCM 工具)
- LLM 调用 MCP tool 前必须 human-in-the-loop 批准(与 cleaner RISKY 计划同理)
- 速率限制: 每个 server 100 req/min
